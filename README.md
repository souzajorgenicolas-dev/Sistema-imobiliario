# Sistema de Gerenciamento Imobiliário

Projeto desenvolvido em **Java** com o objetivo de aplicar os principais conceitos de **Programação Orientada a Objetos (POO)** por meio da implementação de um sistema de gerenciamento de imóveis.

## Funcionalidades

* Cadastro de clientes;
* Cadastro de imóveis (Casa, Apartamento e Terreno);
* Listagem de imóveis;
* Busca de imóveis por código;
* Venda de imóveis;
* Aluguel de imóveis;
* Emissão de contratos;
* Geração de relatórios contendo:

  * Quantidade de imóveis disponíveis;
  * Quantidade de imóveis vendidos;
  * Quantidade de imóveis alugados;
  * Valor arrecadado com vendas;
  * Valor arrecadado com aluguéis;
  * Imóvel de maior valor.

## Conceitos aplicados

* Programação Orientada a Objetos (POO);
* Encapsulamento;
* Herança;
* Polimorfismo;
* Abstração;
* Classes abstratas;
* Interfaces;
* Enumerações (Enum);
* Sobrescrita dos métodos `toString()`, `equals()` e `hashCode()`;
* Tratamento de exceções com exceções personalizadas;
* Coleções utilizando `List` e `ArrayList`.

## Tecnologias

* Java
* Eclipse IDE

## Objetivo

O projeto foi desenvolvido para consolidar conceitos de orientação a objetos, modelagem baseada em diagrama de classes UML e implementação de regras de negócio em Java.
